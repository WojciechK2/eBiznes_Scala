Controller B

Show all (GET method); Path: /controllerB

Controller C

Testowanie:

SHOW ALL: curl -v localhost:9000/controllerA

GET BY ID: curl -v localhost:9000/controllerA/1

UPDATE: curl -vvvv -X PUT -H "Content-Type: application/json" -d '{"value":"ABCD"}' localhost:9000/controllerA/5

DELETE: curl -vvvv -X DELETE localhost:9000/controllerA/5

ADD (POST): curl -vvv -H "content-Type: application/json" -X POST -d '{"value":"nowiutki element"}' localhost:9000/controllerA
